# Zinou's App

A React + TypeScript designer marketplace web app (clients, designers, admin) built with Vite and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Tech stack

- React 19, TypeScript, Vite
- Tailwind CSS, shadcn/ui components
- React Router
